Requirements

  • .NET Framework 4

Installation

  1. Extract everything from the RAR file.
  2. Drag/Drop an xml file on Ryder BXML Converter.exe or use the open with function on an xml file.
  3. See below for further instructions.

If using Vista (not sure if same goes for Windows 7) it may cause errors if you put it in certain folders because of some sort of protection that Vista likes to use. For problems/solutions check below.

Editor Information

This program will allow you to convert any binary xml files in the Dirt/Grid/F1 2012 games.

Features:

  • Complete Support -- Supports converting of all xml types forward and backward.
  • Batch Convert -- Lets you convert multiple binary xml files at once.
  • Open With -- Allows you to choose "Open With" and select this program to convert the xml.

Usage

For the fastest/simplest conversion, Right-Click your xml file, press open with and browse to and select Ryder BXML Converter.exe. This will save the converted file in the same location as your current file with a "C.xml" ending.

You can also convert files by dragging and dropping them on the executable file.

Apart from the above, there are two additional types of arguments you can give to the command line tool:

  1. -c, Conversion Type, File Path, Save Path -- Convert the file to the specified type at the save location.
  2. -b, Conversion Type, File Paths... -- Batch converts multiple xml files to the specified type at its original location with a "C.xml" ending.

There are 4 XmlTypes. Use the numbers when asked for a conversion type by the program:

  • 0 -- Text
  • 1 -- Bin Xml
  • 2 -- BXML Big
  • 3 -- BXML Little

Issues

  • If you get errors try one of the following:
    • Make sure the folder that you put the files in is not readonly.
    • Run the exe as an administrator or give it administrator rights.
    • Put the files in a folder on the desktop.
