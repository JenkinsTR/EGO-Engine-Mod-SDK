Requirements

  • .NET Framework 4

Installation

  1. Extract everything from the 7zip file.
  2. Open Ego TNG Editor.exe.
  3. Enjoy! :)

If using Vista/Windows 7/8 it may cause errors if you put it in certain folders because of some sort of protection that Vista likes to use. For problems/solutions check below.

Editor Information

With this program you can edit TNG files for the EGO engine games. These files contain car setup information.

Features:

  • Intuitive GUI -- Easy to use graphics user interface for editing files.
  • Open With -- Allows you to choose "Open With" and select this program to open TNG files.

Usage

After turning on the program, simply select File -> Open and select a TNG file to edit.

Issues

  • If you get errors try one of the following:
    • Make sure the folder that you put the files in is not readonly.
    • Run the exe as an administrator or give it administrator rights.
    • Put the files in a folder on the desktop.
